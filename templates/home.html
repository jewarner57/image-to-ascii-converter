What, How, Why?

What is ASCII art?

  • ASCII art consists of an image created using only text characters.

    You can learn more about ASCII art here.

How does it work?

  • ASCII art is created by breaking down an image into groups of pixels. Each of those pixels is then converted into a character to be put strung back together into an image.

    Find out more about how ASCII art is made here.

What's the point?

  • ASCII art was used in the early days of computers to depict images when the graphics power to draw actual images wasn't avaliable. ASCII art is a fun throwback to the origins of computers, and it has a distinct, fun look.

    You can learn more about the history of ASCII art here.
